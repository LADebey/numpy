# Piscine IA - Quest 01 : Fondamentaux de NumPy

Ce projet fait partie du cursus **Zone 01**. L'objectif est de maîtriser les bases de la manipulation de données avec **NumPy**, l'outil incontournable de l'écosystème IA en Python.

## 🚀 Objectifs d'apprentissage

- Configuration d'environnements virtuels isolés.
- Manipulation de tableaux multidimensionnels (`ndarray`).
- Maîtrise du **Slicing** (extraction de données sans boucles `for`).
- Utilisation de la **Vectorisation** pour optimiser les calculs.
- Génération de données aléatoires et statistiques de base.

## 🛠️ Installation & Configuration

### Prérequis

- Python 3.9 ou supérieur
- Pip (gestionnaire de paquets Python)

### Mise en place de l'environnement

1. Création de l'environnement virtuel :
   ```bash
   python -m venv ex00
   ```
